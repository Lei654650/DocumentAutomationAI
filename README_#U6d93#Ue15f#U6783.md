# Document Automation AI V7.1.1

## 启动方法

1. 确认工程完整路径中同时存在 `backend` 和 `frontend` 文件夹。
2. 双击根目录的 `Start_All.bat`。
3. 第一次运行会自动创建 Python 虚拟环境并安装前后端依赖，可能需要几分钟。
4. 启动器会等待后端 8000 端口和前端 5173 端口真正就绪后，才自动打开浏览器。
5. 运行期间不要关闭 Backend 和 Frontend 两个窗口。

默认管理员密码：`admin123456`

## 重要说明

不要把新工程解压到旧工程内部。正式路径应为：

`D:\Develop\WebProjects\DocumentAutomationAI`

并且该目录下一层直接看到：

- `backend`
- `frontend`
- `Start_All.bat`
